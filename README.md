# 프로젝트 기능



* 회원가입
  * 정규식 통해 이메일을 가려내기
  * 유저네임에서 리플레이스 사용해서 공백을 가려냄
  * 패스워드랑 패스워드 체크로 다를때 회원가입 방지

* 로그인

* 사진업로드 
  * 자동으로 사물인식 된 사진업로드 해줌


* 머신러닝, Django 연동
  * 업로드 된 사진의 주소를 YOLO 인식 사진으로 바꿔줌
